# 🚀 How To Run

## Backend
```bash
cd backend
python manage.py runserver
```

## Frontend
```bash
cd frontend
npm run dev
```

---

# 🌸 Database

SQLite database file:

```txt
backend/db.sqlite3
```

Products are populated using:

```bash
python manage.py populate_bouquets
```

# 🔥 APIs

## Get Products



```txt
POST /api/users/register/
```

## Cart
